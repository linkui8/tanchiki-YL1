# Танчики на pygame!


Я попытался сделать игру Танчики на компьютер с помощью Python и библиотеки PyGame.

### Как запустить?

У вас должны быть установлены Python и PyGame.

```
pip install -r requirements.txt
```

Запускайте:

```
python main_menu.py
```
